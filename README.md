### Comando para generar QR de proyectos

```http
  npm run GQR
```

Las imagenes Qr se guardan en public/img/qr

### Comando para generar pdf del CV

```http
  npm run GPDF
```

Los pdf de las vistas se guardan en public/docs de config/prod/URLWEB:
